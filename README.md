# IT Top — электронный дневник 📱

Мобильное приложение для студентов **Top Academy / IT Step**. Смотрите
расписание, оценки, домашние задания и свой профиль из веб-журнала прямо
на телефоне — без браузера.

> ⚠️ Неофициальное приложение. Работает с вашим личным аккаунтом веб-журнала.

---

## ✨ Что умеет

- 👤 **Профиль** — фото, группа, поток, уровень, достижения и баллы.
- 📅 **Расписание** — занятия по дням, с переключением даты.
- 📊 **Оценки** — оценки по каждому предмету, средний балл и посещаемость.
- 📝 **Домашние задания** — список по статусам: надо сделать, на проверке,
  проверено, просрочено.
- 🔐 **Удобный вход** — «запомнить пароль» и автоматический вход.
- 🌙 Тёмная тема.

---

## 📥 Установка (Android)

1. Откройте раздел **[Releases](https://github.com/evansvl/it-top-diary/releases/latest)**
   и скачайте файл **`it-top.apk`**.
2. Откройте скачанный файл на телефоне.
3. Если появится предупреждение — разрешите **установку из неизвестных
   источников** (Android сам предложит перейти в настройки).
4. Нажмите «Установить» и откройте приложение.

> 📱 Поддерживается только **Android**. iPhone — нет.

---

## 🔑 Как войти

Введите **логин и пароль от веб-журнала** — те же, что на
[journal.top-academy.ru](https://journal.top-academy.ru).

Включите **«Запомнить пароль»** или **«Автоматический вход»**, чтобы
не вводить данные при каждом запуске.

---

## ❓ Частые вопросы

**Это безопасно?**
Логин и пароль хранятся только на вашем телефоне (в зашифрованном
хранилище Android) и отправляются напрямую на сервер журнала — больше
никуда.

**Откуда берутся данные?**
Приложение показывает то же, что и официальный веб-журнал, — просто
удобнее на телефоне.

**Не получается войти.**
Проверьте логин и пароль на сайте журнала и наличие интернета.
Сообщение «Неверный логин или пароль» означает неправильные данные.
