УСТАНОВКА

Android

  1. Нажмите «Скачать для Android» — загрузится .apk.
  2. Откройте файл; разрешите установку из этого источника, если телефон попросит.
  3. Готово. Обновления приложение предложит само.

iPhone

  1. Скачайте .ipa (он неподписанный).
  2. Поставьте AltStore или Sideloadly на компьютер.
  3. Подпишите .ipa своим Apple ID и установите на iPhone.
  4. Бесплатная подпись живёт 7 дней — потом просто переподпишите.
