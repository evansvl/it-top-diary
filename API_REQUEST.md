## Нужно от тебя (2 curl'а), чтобы добить auth полностью:

### 1. Профиль студента — /settings/user-info
Открой журнал, вкладка профиля/настроек, и пришли curl запроса,
который отдаёт ФИО, фото и группу. Нужен сам ответ (JSON):
```json
{ "...": "..." }
```
Тогда переключу `USE_USER_INFO = true` и имя/фото/группа подтянутся.

### 2. Обновление токена — refresh
Когда access протухнет (≈6 часов), журнал шлёт запрос на обновление.
Нужен его curl: точный URL, метод и тело (там скорее всего refresh_token).
